import the files os, re, pdfplumber,pandas.
	os: used for handling folders and file paths.
	   functions used here:
	        1. os.listdir() : reads files and folders.
			2. os.path.join() : creates proper file paths.
			3. os.makedirs() : creates folders automatically.
            4. os.path.isdir() : checks whether a path is a folder. Prevents the loss of files.
	re: used for regular expressions and text cleaning. identify patterns in extracted text and repair broken values from PDFs.
       functions used here:
            1. re.sub() :  removes extra spaces that were created in making files.
            2. re.match() : checks the text patterns. 
            3. re.search() : finds the patterns inside strings inside the table. 			
    pdfplumber: used to extract tables from PDF pages. It is the main extraction library here.
	    functions used here:
		    1. pdfplumber.open() : opens PDF files.
			2. page.extract_tables() : extracts table data from pages.(only table data)
	pandas: used for dataframe operations such as cleaning, merging, filtering, and exporting data.
	    functions used here:
		    1. pd.DataFrame() :  creates the table.
			2. pd.merge() : merges the pages together.
			3. drop_duplicates() : removes the repetetive or duplicate rows. 
			4. to_csv() : exports the final csv files.
 
Important functions:

def normalize_text(): compare similar labels consistently.
 This function standardizes text by:
    1. converting to lowercase
    2. removing extra spaces
    3. removing line breaks

def standardize_keys(): ensures all `L 1` and `L 2` values remain consistent across every CSV file.
    A global dictionary called `MASTER_KEYS` stores the first valid version and reuses it everywhere.

def fix_split_cells(): detects the split cells, which are created when pdf extraction, merges them correctly.
    uses regex pattern matching to identify broken text.
	
def normalise_key_cols(): ensures the first two columns are always:
 L1 and L2 that are in the data.

It also handles: blank headers, shifted columns, duplicate column names.

def extract_page(): extracts tables from a single PDF page.

work flow:

1. Extract tables
2. Select the largest table
3. Fix split cells
4. Clean headers
5. Convert to dataframe

def forward_fill_l1(): In many PDF tables, repeated category names are left blank. Hence missing values are filled using the previous category values.

def clean_l2(): cleans l2 on removing extra spaces etc..

def drop_empty_rows(): Removes rows where `L 2` is empty because such rows usually contain noise.

def clean_page(): Combines all cleaning functions into one pipeline.

def merge_pages(): merges all pages into one final dataframe using pd.merge() with L1 and L2 as common keys.

def process_pdf(): the main function that controls the full workflow.

Algorithm:

1. Open PDF
2. Read pages
3. Extract tables
4. Clean data
5. Standardize labels
6. Merge pages
7. Remove duplicates
8. Return final dataframe
